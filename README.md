# Omni Executor: Enterprise Reasoning Engine

Omni Executor is a production-grade reasoning engine and business execution architecture system.

## 🛠️ Key Features
- **Scale-Aware Reasoning:** Adapts logic dynamically to business size.
- **Consulting-Grade:** Outputs modeled after top-tier management consulting.
- **Hybrid Intelligence:** AI reasoning with deterministic rule-based fallback.

## ⚙️ Configuration
The app supports the following secrets:
- `GEMINI_API_KEY`: Required for core AI reasoning.

## 🚀 Deployment
1. Fork this repo to GitHub.
2. Link repo to **Streamlit Community Cloud**.
3. Add your `GEMINI_API_KEY` in the Streamlit Secrets menu.
4. Launch!
